Sight Reading Generator

This it a sight reading tool designed around practicing note identification and music reading. Adjusts the settings below and then click generate to get a new melody. Find more information about the controls at the bottom of the page.

  1. Setting the Range:

    The lowest note and highest note sliders set the lowest and highest note that the generator will pick notes from. It defaults to the range of the guitar but you can change it to practice with other instruments or to limit yourself to focus on practicing in a particular position

    If the lowest note is above the highest note the generator will return empty.

  2. Interval Range:

    You can also set the range of intervals that are used in the melody from easy (step-wise motion), medium (3rds), hard (4ths), and advanced (6ths).

  3. Rhythm Setting:

    The rhythm setting adjust the type of rhythms used. Right now you can select from only quarter notes, only eighth notes, a combination of the two, and the combination with rests (both quarter and eighth note lengths).

  4. Key and Clef:

    Change the key signature or clef using the drop down menus. For most clef changes you will also need to adjust the range to the instrument you are practicing with.
